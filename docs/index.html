🚀 Kube User Admin Helm Repository

Kubernetes RBAC User Management Helm Chart

Welcome to the official Helm chart repository for Kube User Admin - a full-stack application for managing Kubernetes users and permissions.

📦 Quick Installation

# Add the Helm repository
helm repo add kube-user-admin https://renatoruis.github.io/kube-user-admin
helm repo update

# Install the chart
helm install kube-user-admin kube-user-admin/kube-user-admin \
  --namespace kube-user-admin \
  --create-namespace

🔧 Custom Installation

# Install with custom values
helm install kube-user-admin kube-user-admin/kube-user-admin \
  --namespace kube-user-admin \
  --create-namespace \
  --set backend.env.ADMIN_PASSWORD="your-secure-password" \
  --set ingress.enabled=true \
  --set ingress.hosts[0].host="kube-admin.yourdomain.com"
